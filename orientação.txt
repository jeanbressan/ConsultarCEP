Título da Atividade: 
Explorando a API Fetch para Consulta de CEP e Local Storage

Descrição da Atividade:
Nesta atividade, você terá a oportunidade de aplicar seus conhecimentos em HTML, CSS e JavaScript para criar uma aplicação web interativa. Seu objetivo é desenvolver uma tela onde o usuário possa inserir um CEP e, ao clicar em um botão, consultar uma API de CEP para obter os dados do endereço correspondente e exibi-los na tela. Além disso, você também deve implementar a funcionalidade de salvar os dados do endereço no Local Storage do navegador.

Instruções:
1. Desenvolva uma interface HTML com CSS que inclua:
   - Um campo de entrada para o usuário digitar o CEP.
   - Um botão para acionar a consulta à API de CEP.
   - Uma área na qual os dados do endereço serão exibidos após a consulta.
   - Um botão para salvar os dados do endereço no Local Storage.

2. Utilize JavaScript para:
   - Capturar o evento de clique no botão de consulta à API de CEP.
   - Fazer uma requisição utilizando a API Fetch para a API de CEP (por exemplo, ViaCEP).
   - Tratar a resposta da API e exibir os dados do endereço na área designada.
   - Implementar a funcionalidade de salvar os dados do endereço no Local Storage ao clicar no botão correspondente.

3. Certifique-se de que, ao recarregar a página, a lista de endereços salvos no Local Storage seja exibida.
